# RunRun
 
스파르타 던전 탐험 만들기🎮

황금 사과를 찾아서 Escape와 점프맵을 합친 게임을 제작했습니다.


## 개요

* 프로젝트 이름 : RunRun
* 프로젝트 기간 : 2025.03.06 - 2025.03.11
* 개발 엔진, 언어 : 유니티, C#
* 개인 프로젝트
